- #### プロジェクト規模
  10〜15 人チームでの開発
- #### 役割
  設計、実装、テスト、レビュー
- #### プロジェクト詳細
  - Laravel（BFF として使用）でのヘッドレスな EC フロント開発
  - 社内で運用されている EC フロント用 SDK の開発（PHP)

  - CI/CD パイプラインの改善、静的解析ツール（PHPStan）の導入
  - XSS 脆弱性への対応
  - フロントエンドはノンフレームワークな Vanilla JS を使用
